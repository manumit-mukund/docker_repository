Steps to run a java file on Docker on Window 10 Pro (example given below):

1. Create folder, e.g.,  C:\mygitrepositories\docker_repository\mydockerexamples\javaexamples
2. Create a file, e.g., HelloWorld.java in the above folder
3. Create a docker file, Dockerfile, without any extension in the above folder
4. Start Desktop Docker (download it if not available on your machine, e.g, Docker version 26.1.4, build 5650f9b)
5. Open Command Prompt (Run as administrator) and run the following commands
6. cd C:\mygitrepositories\docker_repository\mydockerexamples\javaexamples
7. docker build -t helloworld_docker_image .
8. docker run helloworld_docker_image
